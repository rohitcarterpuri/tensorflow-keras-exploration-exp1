# TensorFlow and Keras Basic Features Exploration

A comprehensive guide and implementation exploring the basic features of TensorFlow and Keras packages for deep learning.

## 📋 Overview

This project demonstrates key features of TensorFlow and Keras including:
- Basic tensor operations and automatic differentiation
- Building neural networks with Sequential API
- Convolutional Neural Networks (CNNs) for image classification
- Model training, evaluation, and visualization
- Model saving and loading capabilities
